# apex-smart-password-plugin
APEX Smart Password Plugin is a custom Oracle APEX plugin that helps enforce strong password policies with real-time validation and user-friendly feedback. It improves security by guiding users to create strong passwords and preventing weak or common passwords during registration or password change.




# Smart Password Strength Plugin for Oracle APEX

This plugin checks password strength in Oracle APEX.

## Features
- Weak / Medium / Strong password detection
- Real-time validation
- Easy install

## Installation
1. Download SQL file
2. Import into APEX (Shared Components → Plugins → Import)
3. Use item type in page
